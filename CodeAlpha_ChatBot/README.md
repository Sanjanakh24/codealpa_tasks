# CodeAlpha_ChatBot

A simple web-based chatbot built using **Python Flask**, **HTML**, **CSS**, and **JavaScript**. The chatbot uses rule-based logic to respond to user queries related to services, internships, cloud computing, and other predefined topics.

## About

CodeAlpha_ChatBot is a simple interactive chatbot application designed to provide informative responses to user questions. It is implemented using Python Flask on the backend and standard web technologies on the frontend.

---

## Features

- Rule-based response logic for predefined queries  
- Web interface for user interaction  
- Integration between Flask backend and frontend UI  
- Lightweight and easy to extend with additional responses

## Installation

1. **Clone the repository**
  ```bash
   git clone https://github.com/Sanjanakh24/CodeAlpha_ChatBot.git
   cd CodeAlpha_ChatBot
 ````


2. **Create a virtual environment (optional but recommended)**

   ```bash
   python3 -m venv venv
   source venv/bin/activate        # macOS / Linux
   .\venv\Scripts\activate         # Windows
   ```


3. **Install dependencies**

   ```bash
   pip install -r requirements.txt
   ```


---

## Usage

1. **Run the Flask application**

   ```bash
   python app.py
   ```

2. **Open your browser**

   ```
   http://127.0.0.1:5000
   ```

3. **Start chatting**
   Enter questions in the chatbot interface to receive responses.
